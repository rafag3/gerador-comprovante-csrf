Gerador de Comprovantes CSRF

Automação em Python que gera comprovantes anuais de retenção
de CSLL, COFINS e PIS/PASEP a partir de uma planilha Excel.

Funcionalidades

✔ Leitura automática de planilha Excel
✔ Consolidação de pagamentos por fornecedor
✔ Geração de comprovantes em PDF
✔ Compactação automática em arquivo ZIP
✔ Interface gráfica para uso interno

Tecnologias utilizadas

- Python
- Pandas
- ReportLab
- CustomTkinter
